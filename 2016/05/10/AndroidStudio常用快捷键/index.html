AndroidStdio常用快捷键

发表于 2016-05-10   |   分类于 AndroidStudio   |  

总结一下AndroidStudio常用的一些快捷键，针对Windows。

  • Ctrl + Alt + L 代码格式化
  • Ctrl + Alt + Space 代码提示
  • Ctrl + Shift + up/down 上下行移动代码行/块
  • Ctrl + Alt + left/right 跳转到光标的上一次/下一次的位置
  • Ctrl + O 重写父类方法
  • Ctrl + Shift + I 查看一个方法的实现
  • Alt + left/right 切换打开文件窗口
  • Ctrl + F4 关闭当前标签
  • Shift + ESC 关闭打开的Tool Window,编辑器或取焦点
  • Ctrl + +/- 代码折叠展开
  • Alt + 1 打开隐藏工程面板
  • Alt + F7 Find Usage
  • Ctrl + Shift + Alt + N 查找方法和变量
  • Ctrl + F12 显示文件内部结构
  • Ctrl + [/]跳转到方法的开始和结束{}
  • Ctrl + Alt + T Surround With
  • Ctrl + B 查看变量声明
  • Ctrl + 左键 查看变量声明
  • Ctrl + J 自动生成模板代码(Toast,Log等)
  • Ctrl + N 查找一个类
  • Ctrl + E Recent Files
  • Shift + Shift Search Everywhere
  • Ctrl + U 查看父类
  • Ctrl + W 选择一行代码，再按一次选择一个方法体
  • Ctrl + D 复制当前行到下一行
  • Ctrl + H 查看类继承结构
  • Ctrl + Alt + H 查看一个方法的调用

打印消息

  • logd
  • loge
  • logi
  • logm
  • logr
  • logw

调试

  • F8 单步调试
  • F7 进入方法内部
